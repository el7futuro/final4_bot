# 🎮 Final 4 Bot - Telegram бот для настольной игры

## Описание
Telegram бот для игры Final 4 - спортивной настольной игры с кубиками и карточками.

## Основные функции
- Управление командой из 16 футболистов
- Матчи против реальных игроков и ботов
- Система ставок на кубик
- Карточки "Свисток" (40 типов)
- Турниры и рейтинговая система
